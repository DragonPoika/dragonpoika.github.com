Here's some extra info for you on the tool:
Field meanings:
Namespace:
The pack's namespace. You should always use one if you're adding custom content!
Base Item ID:
Whatever item you use as the base item. Vanilla items!
Custom Model Data:
Quite self-explanatory. Item's Custom Model Data.
Custom Item ID:
Your custom item's ID. Used in the translation string, what tag is given to the item, and the downloaded file's name.
Pack Color:
The color of the pack. Used in the lore of the item.
Translation Fallback:
The translation fallback. The translation string is automatically generated with 'item.namespace.customitemid'

Note: This website assumes you have a custom icon on your pack tooltip with the font name "namespace:tooltip". Maybe someday that can be disabled.
But for now, just remove the "set_lore" function's first two json strings!
